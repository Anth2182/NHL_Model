📋 Project Overview

A probabilistic NHL game prediction system that uses a Dixon-Coles Poisson model with Skellam distribution to generate win probabilities, puck line coverage, and over/under probabilities for every NHL game. The model blends prior season ratings, current season performance, and advanced metrics (xGoals, high-danger chances, shots on goal) with goalie quality adjustments based on Goals Saved Above Expected (GSAx).

⚙️ Nightly Automated Workflow

1

Scrape Game Logs

GitHub Actions runs scrape_gamelogs.py. Downloads the latest game-by-game data from MoneyPuck and appends any new games to the NHL_Gamelog table in Supabase.

2

Scrape Goalie Stats

Runs scrape_goalies.py. Downloads current season goalie stats from MoneyPuck and replaces the Goalie_stats table in Supabase with fresh data.

3

Retrain the Model

Runs train_model.py. Pulls all game and goalie data from Supabase, fits new Dixon-Coles attack/defense ratings via MLE, and saves the updated nhl_skellam_model.json.

4

Commit Updated Model

GitHub Actions automatically commits the new nhl_skellam_model.json back to the repository so the latest model is always available.

Schedule: Runs every night at 11:00 PM EST automatically. No computer needs to be on — everything runs on GitHub's cloud servers. Can also be triggered manually from the GitHub Actions tab at any time.
